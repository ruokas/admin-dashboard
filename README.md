# Traumos forma – GitHub Pages versija

Šiame repozitorijoje yra statinis `index.html` failas.

## GitHub Pages aktyvavimas
1. GitHub'e atidaryk repozitorijos **Settings**.
2. Skiltyje **Pages** pasirink šaltinį: `main` šaka ir root (`/`).
3. Išsaugok – po kelių minučių puslapis bus pasiekiamas per `https://<tavo-vartotojas>.github.io/<repo-pavadinimas>/`.

## Vietinis paleidimas
Tiesiog atidaryk `index.html` naršyklėje.
